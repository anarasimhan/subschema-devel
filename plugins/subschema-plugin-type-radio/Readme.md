Renders radio groups. Options can be strings or
 {val, label} objects.

```js static
{
  "type":"Radio",
  "options": [
        {
          "val": 0,
          "label": "Option 1"
        },
        {
          "val": 1,
          "label": "Option 2"
        },
        {
          "val": 2,
          "label": "Option 3"
        }
      ]
}
```
      
See the [example](http://subschema.github.io/subschema/#/Radio)
